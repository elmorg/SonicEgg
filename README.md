SonicEgg
========

Audio spectrogram tool

3D Spectrogram with Microphone Input

User controls:

press 'p' to pause view
press 'v' to toggle view
press 't' to toggle freq labels
press 'f' to toggle fade effect
press 'd' to toggle dB amplitude scale
